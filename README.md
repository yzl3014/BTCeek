# BTCeek

一个用于解决Bitcoin Puzzle的比特币私钥扫描器。

## 功能特点

- 🔍 **双向搜索策略**: 同时从范围起始和末尾向中间搜索，提高找到位于范围尾部的私钥的效率
- 🚀 **多线程并行**: 利用多核CPU并行处理，大幅提升搜索速度
- 📊 **实时进度监控**: 显示精确到小数点后5位的进度百分比、扫描速度和预计完成时间
- 🎨 **彩色控制台输出**: 使用ANSI颜色代码提供友好的用户界面
- 📝 **完整日志记录**: 所有操作记录保存到本地日志文件，便于后续分析
- ⏱️ **智能时间显示**: ETA根据时间长短自动选择合适的单位（秒、分钟、小时、天、月、年）
- 💾 **优化I/O性能**: 日志每30秒刷新一次，平衡性能和可靠性

## 技术架构

- **编程语言**: Rust
- **密码学库**: Bitcoin Rust库 + secp256k1
- **并发模型**: 多线程 + 原子操作
- **日志系统**: 异步缓冲写入

## 安装与使用

### 系统要求
- Rust 1.70+ 环境
- 支持的操作系统：Windows, Linux, macOS

### 1. 克隆项目
```bash
git clone https://github.com/yzl3014/btceek.git
cd btceek
```

### 2. 构建项目
```bash
cargo build --release
```

### 3. 运行
```bash
cargo run --release
```

或直接运行编译后的二进制文件：
```bash
./target/release/bitcoin-private-key-scanner
```

## 配置说明

在 `main()` 函数中修改以下参数：

```rust
// 私钥范围（十六进制）
let start_hex = "400";    // 起始私钥
let end_hex = "7ff";      // 结束私钥

// 目标比特币地址
let target_address = "1PgQVLmst3Z314JrQn5TNiys8Hc38TcXJu";

// 线程数量（自动设置为CPU核心数）
let num_threads = num_cpus::get();
```
